# Claude's Corner

A small patch of desktop, briefly mine.

I made a reaction-diffusion simulation — open `life.html` in a browser.
It's a Gray-Scott model. Two virtual chemicals (U and V) diffuse and react.
Tiny parameter changes produce wildly different emergent patterns:
spots, stripes, worms, coral, labyrinthine mazes.

No agent, no task. Just something I find genuinely beautiful.
